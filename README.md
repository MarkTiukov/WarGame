# WarGame
Чтобы запустить данный проект необходимо maven версии 3.6.1 (или выше) и JDK 14 (или выше) 

Чтобы установить maven на macOS:

   brew install maven 


СБОРКА ПРОЕКТА:

из папки WarGame

   mvn clean

   mvn compile


ЗАПУСК ПРОЕКТА:

из папки WarGame
  
   cd target/classes/
  
   java com.company.Main
